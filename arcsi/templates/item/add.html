Broadcast

Airing an episode on a radio show's time slot has 2 forms. 1, Live - Image uploads are allowed, audio uploads are disabled. 2, Recorded - Image and music uploads are mandatory.

The current episode airing on the station is reset in both cases.

When selecting the default choice ("Skip") the episode is archived without updating the station.

On the inverse, airing an episode also archives that episode by default. This setting can be turned off on a show-level currently.

The station schedule rotates weekly. If any broadcasting type is selected the episode airs the next time the show is broadcasted. If no episode is added during the week then the show airs the most recent episode by default.

Setting the date of airing is optional, when no date is set the default date is tomorrow - relatively, i.e. the last possible submission is the day before show airs. The schedule is reloaded each dawn.

Summary: When an episode is ready submit it within one week before scheduled broadcast time and select from the date-picker the day the show airs.
